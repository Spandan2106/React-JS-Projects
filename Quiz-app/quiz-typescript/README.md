## ‚ú® Features

- **Dynamic Timed Questions**: Each question comes with a 10-second timer to keep you on your toes.
- **Interactive Question Navigation**: A navigation matrix allows you to jump to any question at any time.
- **Answer Tracking**: The navigation matrix visually tracks which questions have been answered and which are still pending.
- **Submit Confirmation**: A confirmation dialog prevents accidental submissions, showing you how many questions are left unanswered.
- **Dynamic Scoring**: Your final score is calculated and displayed upon completion.
- **Conditional Results**: Receive a congratulatory message for scores above 50% and an encouraging one for scores below.
- **Answer Review**: After finishing the quiz, you can review all the questions and their correct answers.
- **Engaging UI**: A beautiful, animated gradient background with space-themed elements like a solar system, black hole, and shooting stars, all created with CSS.
- **Type-Safe Code**: Built with TypeScript for a robust and maintainable codebase.

## üöÄ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm installed on your machine.
- Node.js
- npm

### Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. **Navigate to the project directory:**
   ```sh
   cd your-repo-name
   ```

3. **Install NPM packages:**
   ```sh
   npm install
   ```

4. **Run the application:**
   ```sh
   npm start
   ```
   The app will open automatically in your browser at `http://localhost:3000`.

## üéÆ How to Play

1. Answer each question by clicking on one of the options. The quiz will automatically move to the next question.
2. Keep an eye on the timer! If it runs out, you'll be moved to the next question without answering.
3. Use the **Navigate Questions** button to see your progress and jump to any question.
4. When you're ready, click **Submit Quiz** to see your final score and review the correct answers.
5. Click **Restart Quiz** at any time to start over.
