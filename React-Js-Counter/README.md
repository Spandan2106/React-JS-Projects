## üõ†Ô∏è Technologies Used

This project demonstrates how to run a React application without a complex setup by leveraging CDNs and in-browser transpilation.

-   **HTML5**
-   **CSS3**: For all styling, layout, and animations.
-   **JavaScript (ES6+)**
-   **React**: Loaded via a CDN to manage the UI components and state.
-   **ReactDOM**: Loaded via a CDN to render the React components into the DOM.
-   **Babel Standalone**: Used for in-browser transpilation of JSX syntax into plain JavaScript that the browser can understand.

This approach is excellent for small projects, prototypes, and for learning React without the overhead of setting up a full development environment like Vite or Create React App.
